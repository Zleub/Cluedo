_findSegmentForLocation(location, absolute)
returns [segment, proportion of travel in segment, segment index] for the segment that contains the point which is 'location' distance along the entire path, where 'location' is a decimal between 0 and 1 inclusive. in this connector type, paths are made up of a list of segments, each of which contributes some fraction to the total length. From 1.3.10 this also supports the 'absolute' property, which lets us specify a location as the absolute distance in pixels, rather than a proportion of the total path.
